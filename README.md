# Pixpress

Pixpress is an AI-powered image caption generator web application.

Users can upload an image and the system automatically generates a caption
using a deep learning model.

## Features
- Upload an image
- AI caption generation
- Text-to-speech caption playback
- Clean responsive UI

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

AI Model:
- BLIP2 image captioning model

## How it works
1. User uploads an image
2. Image is sent to the Flask backend
3. BLIP2 model generates a caption
4. Caption is displayed on the webpage
